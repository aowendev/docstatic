Par défaut, DocStatic est configuré pour utiliser la recherche [Lunr](https://lunrjs.com/). Son principal avantage est qu'il s'agit d'une solution gratuite. Son principal inconvénient est que l'intégralité de l'index de recherche est téléchargée sur le navigateur de l'utilisateur ; plus votre site est volumineux, plus il est lent.

Diverses alternatives sont disponibles sous forme de plugins Docusaurus, notamment :

* Algolia (préinstallé)
* [Markprompt](https://github.com/typesense/docusaurus-theme-search-typesense)
* [Meilisearch](https://github.com/tauri-apps/meilisearch-docsearch)
* [Orama](https://github.com/askorama/orama/tree/main/packages/plugin-docusaurus-v3)
* [Typesense DocSearch](https://github.com/typesense/docusaurus-theme-search-typesense)

Pour plus d'informations, consultez la section [Recherche](https://docusaurus.io/docs/search#using-algolia-docsearch) dans la documentation Docusaurus.
