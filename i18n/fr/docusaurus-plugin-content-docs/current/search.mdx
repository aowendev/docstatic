Par défaut, DocStatic est configuré pour utiliser la recherche [Lunr](https://lunrjs.com/). Son principal avantage est qu'il s'agit d'une solution à coût nul. Son principal inconvénient est que l'index de recherche complet est téléchargé dans le navigateur de l'utilisateur ; plus votre site est volumineux, plus il est lent.

Diverses alternatives sont disponibles sous forme de plugins Docusaurus, notamment :

* Algolia (pré-installé)
* [Markprompt] (https://github.com/typesense/docusaurus-theme-search-typesense)
* [Meilisearch](https://github.com/tauri-apps/meilisearch-docsearch)
* [Orama](https://github.com/askorama/orama/tree/main/packages/plugin-docusaurus-v3)
* [Typesense DocSearch](https://github.com/typesense/docusaurus-theme-search-typesense)

Pour plus d'informations, voir [Search](https://docusaurus.io/docs/search#using-algolia-docsearch) dans la documentation de Docusaurus.
