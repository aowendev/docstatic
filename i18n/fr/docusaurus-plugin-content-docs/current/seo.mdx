DocStatic soutient l'optimisation des moteurs de recherche de différentes manières :

* Métadonnées globales
* Liens lisibles par l'homme
* Méta description de l'image
* Informations de recherche riches
* Fichier robots
* Métadonnées de page unique
* Fichier Sitemap
* Contenu structuré

Pour plus d'informations, reportez-vous à la section [Optimisation des moteurs de recherche](https://docusaurus.io/fr/docs/seo) dans la documentation de Docusaurus.
