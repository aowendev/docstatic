DocStatic prend en charge l'optimisation pour les moteurs de recherche de différentes manières :

* Métadonnées globales
* Liens lisibles par l'utilisateur
* Méta-description des images
* Informations de recherche enrichies
* Fichier robots
* Métadonnées de page unique
* Fichier sitemap
* Contenu structuré

Pour plus d'informations, consultez la section [Optimisation pour les moteurs de recherche](https://docusaurus.io/docs/seo) dans la documentation Docusaurus.
