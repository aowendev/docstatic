DocStatic vous permet de définir la liste des navigateurs pris en charge au moyen d'une configuration [liste des navigateurs](https://github.com/browserslist/browserslist)). En règle générale, les sites web devraient correctement fonctionner sur n'importe quel navigateur moderne.

Pour plus d'informations, voir [Navigateurs pris en charge](https://docusaurus.io/fr/docs/browser-support) dans la documentation de Docusaurus.

## Édition dans le CMS

TinaCMS est entièrement compatible avec :

* Chrome
* Edge
* Safari

<Admonition type="danger" title="Avertissement">
  TinaCMS a des problèmes connus avec Firefox qui peuvent aller de l'ignorance des styles de caractères à la non sauvegarde du travail. Vous devriez utiliser un autre navigateur.
</Admonition>
