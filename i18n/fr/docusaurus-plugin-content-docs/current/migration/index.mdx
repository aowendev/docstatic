# Mise à niveau

DocStatic étant encore en phase de développement précoce, les modifications doivent être fusionnées individuellement à partir du référentiel principal. Cependant, les composants Node.js peuvent être mis à jour en toute sécurité en remplaçant le fichier [package.json](https://github.com/aowendev/docstatic/blob/main/package.json) par la dernière version du référentiel principal. Après la mise à jour, exécutez :

```shell
yarn install
```
