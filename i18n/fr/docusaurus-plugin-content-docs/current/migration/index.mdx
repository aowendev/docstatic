# Mise à niveau

DocStatic étant encore au début de son développement, les modifications doivent être fusionnées individuellement à partir du dépôt principal. Cependant, les composants Node.js peuvent être actualisé en toute sécurité en remplaçant le fichier [package.json](https://github.com/aowendev/docstatic/blob/main/package.json) par la dernière version du dépôt principal. Après la mise à jour, exécutez :

```shell
yarn install
```
