# Créer un document

Dans DocStatic, vous rédigez votre contenu au format Markdown dans des fichiers appelés « Topics » (sujets). En règle générale, vous créez vous-même chaque sujet, bien que certains types de documents puissent comporter des sujets générés automatiquement.

<Admonition type="note" title="Remarque">
  Les documents GraphQL et OpenAPI sont générés automatiquement à partir d'un serveur ou d'un schéma.
</Admonition>

Pour créer un sujet :

1. Dans la collection **Topics**, cliquez sur **Add File**.
2. Donnez un **Title** au sujet.
3. (Facultatif) Si vous souhaitez utiliser le sujet dans l'[aide contextuelle](https://docstatic.com/docs/context-help), donnez-lui un **Slug**. Il s'agit d'un chemin d'accès fixe à partir de `/docs/`.
4. Donnez une **Description** au sujet. Celle-ci s'affiche dans la liste des fiches de documentation.
5. (Facultatif) Ajoutez les **balises** [taxonomiques](taxonomies) requises.
6. (Facultatif) Définissez les **conditions**.
7. Assurez-vous que le statut du workflow est défini sur **Brouillon**.
8. Saisissez votre contenu.
9. Cliquez sur **Enregistrer**.

## Avant-propos

Au début de chaque rubrique se trouve une section YAML contenue entre deux séries de trois tirets (---). DocStatic l'utilise pour des éléments tels que :

* Texte conditionnel
* Taxonomies (balises de document)
* Workflows

Pour plus d'informations, consultez [Créer un document](https://docusaurus.io/docs/create-doc) dans la documentation Docusaurus.
