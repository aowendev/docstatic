# Introduction à la documentation

La fonctionnalité de documentation vous permet d'organiser les fichiers Markdown dans un format hiérarchique.

<Admonition type="info" title="Info">
  Reportez-vous à la section [Docs Plugin API Reference](https://docusaurus.io/docs/api/plugins/@docusaurus/plugin-content-docs) de la documentation Docusaurus pour obtenir la liste complète des options.
</Admonition>

Pour plus d'informations sur la configuration des modes « docs-only » (documents uniquement) ou « blog-only » (blog uniquement), consultez la section [Introduction aux documents](https://docusaurus.io/docs/docs-introduction) de la documentation Docusaurus.
