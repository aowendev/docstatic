# Docs introduction

La fonctionnalité docs permet d'organiser les fichiers Markdown dans un format hiérarchique.

<Admonition type="info" title="Info">
  Reportez-vous à [Docs Plugin API Reference](https://docusaurus.io/docs/api/plugins/@docusaurus/plugin-content-docs) dans la documentation de Docusaurus pour une liste exhaustive des options.
</Admonition>

Pour plus d'informations sur la configuration des modes “docs-only” ou “blog-only”, voir [Introduction des docs](https://docusaurus.io/fr/docs/docs-introduction) dans la documentation de Docusaurus.
