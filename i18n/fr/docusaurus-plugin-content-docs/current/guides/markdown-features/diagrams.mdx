# Diagrammes

Les diagrammes peuvent être rendus à l'aide de [Mermaid](https://mermaid.js.org/) dans un bloc de code.

## Utilisation

Le CMS prend directement en charge l'édition des diagrammes Mermaid.

1. Cliquez sur l'icône **Mermaid**.
2. Entrez le Markdown dans la boîte.

Cliquez sur **Aperçu** pour afficher votre diagramme.

#### Exemple :

```mermaid
graph TD;
    A-->B;
    A-->C;
    B-->D;
    C-->D;
```

<Admonition type="note" title="Remarque">
  Actuellement, les diagrammes zenUML ne sont pas pris en charge et les diagrammes de cartes mentales peuvent produire un comportement inhabituel. Tous les types de diagrammes ne peuvent pas être prévisualisés dans le CMS.
</Admonition>

Pour plus d'informations, consultez [Mermaid](https://tina.io/docs/reference/rich-text-usage/mermaid) dans la documentation TinaCMS.

Pour plus d'informations sur les thèmes Mermaid, consultez [Diagrams](https://docusaurus.io/docs/markdown-features/diagrams) dans la documentation Docusaurus.
