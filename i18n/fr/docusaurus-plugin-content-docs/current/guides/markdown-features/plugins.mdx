# Plugins MDX

MDX dispose d'un [système de plugins](https://mdxjs.com/advanced/plugins/) intégré qui peut être utilisé pour personnaliser la manière dont les fichiers Markdown sont analysés et transformés en JSX. Cependant, si vous modifiez le comportement de Markdown, vous devez également refléter ces modifications dans le CMS.

<Admonition type="danger" title="Avertissement">
  Les plugins MDX qui modifient le comportement des composants Markdown standard peuvent nuire à la compatibilité avec le CMS. Procédez à vos propres risques.
</Admonition>

Pour plus d'informations, consultez la section [Plugins MDX](https://docusaurus.io/docs/markdown-features/plugins) de la documentation Docusaurus.

<Admonition type="note" title="Remarque">
  docStatic préinstalle et configure les [plugins mathématiques](math-equations) pour vous.
</Admonition>
