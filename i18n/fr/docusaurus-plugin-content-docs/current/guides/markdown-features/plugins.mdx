# Plugins MDX

MDX dispose d'un [système de plugins](https://mdxjs.com/advanced/plugins/) intégré qui peut être utilisé pour personnaliser la façon dont les fichiers Markdown sont analysés et transformés en JSX. Cependant, si vous modifiez la manière dont Markdown se comporte, vous devez également refléter ces changements dans le CMS.

<Admonition type="danger" title="Avertissement">
  Les plugins MDX qui modifient le comportement des composants Markdown standard peuvent rompre la compatibilité avec le CMS. Procédez à vos risques et périls.
</Admonition>

Pour plus d'informations, voir [Plugins MDX](https://docusaurus.io/fr/docs/markdown-features/plugins) dans la documentation de Docusaurus.

<Admonition type="note" title="Note">
  DocStatic préinstalle et configure les [plugins mathématiques](math-equations) pour vous.
</Admonition>
