# Commentaires

Le composant **Commentaire** vous permet d'ajouter des commentaires qui sont visibles dans le CMS, mais pas sur la page rendue, même dans la source brute.

1. Sélectionnez **Commentaire** dans la liste **Intégrer**.
2. Modifiez le composant.
3. Entrez un **Réviseur**.
4. Entrez un **Commentaire**.

#### Exemple

<Comment comment="Ce commentaire est invisible sur le site web publié." reviewer="AOwenDev" />
