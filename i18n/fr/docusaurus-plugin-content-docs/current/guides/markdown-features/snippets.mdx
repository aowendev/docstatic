# Extraits

Le composant **Extrait** vous permet de réutiliser du contenu dans l'ensemble de votre documentation. Un extrait est un morceau de contenu qui peut être inséré dans une ou plusieurs rubriques. Lorsque vous apportez des modifications à l'extrait, celles-ci sont répercutées partout où l'extrait est utilisé. Cela peut être particulièrement utile pour les textes standardisés. Les extraits ont leur propre collection dans le CMS et sont contenus dans une partie distincte du référentiel.

1. Sélectionnez le composant **Extrait** dans la liste **Intégrer**.
2. Modifiez le composant.
3. Choisissez un fichier dans la liste. La liste est remplie avec tous les fichiers du dossier `/reuse/snippets/` chaque fois que vous exécutez le serveur de développement ou effectuez une compilation.
