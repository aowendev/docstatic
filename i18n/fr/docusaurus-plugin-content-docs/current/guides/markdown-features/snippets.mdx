# Snippets

Le composant **Snippet** vous permet de réutiliser du contenu dans votre documentation. Un extrait est un morceau de contenu qui peut être inséré dans une ou plusieurs rubriques. Lorsque vous apportez des modifications à l'extrait, elles sont apportées partout où l'extrait est utilisé. Cela peut s'avérer particulièrement utile pour les textes passe-partout. Les snippets ont leur propre collection dans le CMS et sont contenus dans une partie distincte du référentiel.

1. Sélectionnez le composant **Snippet** dans la liste **Embed**.
2. Modifiez le composant.
3. Saisissez le **Path** du snippet (dans le dossier `/reuse/snippets/`).

#### Exemple

<Snippet filepath="example.mdx" />
