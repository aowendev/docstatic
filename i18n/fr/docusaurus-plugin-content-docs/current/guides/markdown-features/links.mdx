# Liens Markdown

Il existe deux façons d'ajouter un lien vers une autre page : via un chemin d'accès URL ou un chemin d'accès fichier.

```
- [Chemin d'accès URL vers un autre document](./installation)
- [Chemin d'accès fichier vers un autre document](./installation.mdx)
```

Dans le CMS, vous pouvez ajouter un lien à partir du presse-papiers en sélectionnant une section de texte et en la collant.

Vous pouvez également sélectionner une partie du texte et cliquer sur l'icône Lien pour ajouter un chemin d'accès.

Vous pouvez également placer le curseur à l'endroit où vous souhaitez que le lien apparaisse, cliquer sur l'icône et saisir le chemin d'accès et le texte d'affichage.

Pour plus d'informations sur les liens, consultez la section [Liens Markdown](https://docusaurus.io/docs/markdown-features/links) dans la documentation Docusuarus.
