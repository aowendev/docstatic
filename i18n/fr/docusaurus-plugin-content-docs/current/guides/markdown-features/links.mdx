# Liens Markdown

Il existe deux façons d'ajouter un lien vers une autre page : par un chemin d'URL ou par un chemin de fichier.

```
- [chemin URL vers un autre document](./installation)
- [chemin d'accès à un autre document](./installation.mdx)
```

Dans le CMS, vous pouvez ajouter un lien à partir du bloc-notes en sélectionnant une section de texte et en la collant.

Vous pouvez également mettre en évidence une section de texte et cliquer sur l'icône Lien pour ajouter un chemin d'accès.

Vous pouvez aussi placer le curseur à l'endroit où vous souhaitez que le lien apparaisse, cliquer sur l'icône et saisir le chemin et le texte d'affichage.

Pour plus d'informations sur les liens, voir [Liens Markdown](https://docusaurus.io/fr/docs/markdown-features/links) dans la documentation Docusuarus.
