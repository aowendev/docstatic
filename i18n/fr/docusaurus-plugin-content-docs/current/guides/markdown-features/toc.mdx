# Titres et table des matières

Dans le CMS, vous pouvez sélectionner le niveau **Paragraphe** ou **Titre** (de 1 à 6).

(jsx\:a id="anchor-example")(id\:anchor-example)(/jsx\:a)

## Identifiants de titre

Les identifiants de titre sont générés automatiquement. Par exemple, `heading-ids`. Vous pouvez les utiliser pour créer un lien direct vers les titres des rubriques. Cependant, si vous modifiez le titre, le lien ne fonctionnera plus. C'est pourquoi DocStatic fournit le composant **Aide contextuelle**. Celui-ci intègre une balise d'ancrage n'importe où sur une page, et pas seulement au niveau d'un titre.

1. Sélectionnez **Aide contextuelle** dans la liste **Intégrer**.
2. Modifiez le composant et attribuez-lui un **ID d'aide contextuelle**.

Pour créer un lien vers l'ancrage, utilisez le chemin d'accès : `<URL>/docs/<slug>#<ID>.`

Par exemple : [https://docstatic.com/docs/guides/markdown-features/toc#anchor-example](https://docstatic.com/docs/guides/markdown-features/toc#anchor-example)

## Niveau des titres dans la table des matières

Chaque document Markdown affiche une table des matières dans le coin supérieur droit. Par défaut, cette table n'affiche que les titres `h2` et `h3`, ce qui devrait suffire pour avoir un aperçu de la structure de la page. Si vous avez besoin de modifier la plage des titres affichés, vous pouvez personnaliser le niveau minimum et maximum des titres, soit par page, soit globalement.

Pour plus d'informations, consultez la section [En-têtes et table des matières](https://docusaurus.io/docs/markdown-features/toc) dans la documentation Docusaurus.
