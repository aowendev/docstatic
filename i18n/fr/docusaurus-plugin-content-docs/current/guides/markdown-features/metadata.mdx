# Métadonnées d'entête

DocStatic définit automatiquement pour vous des métadonnées utiles dans `<html>`, `<head>` et `<body>`.
Il est possible d'ajouter des métadonnées supplémentaires (ou de remplacer les métadonnées existantes) avec la balise `<head>` dans les fichiers Markdown.

Dans le CMS, ces métadonnées doivent être ajoutées dans l'éditeur Markdown. Elle ne sera pas reconnue si vous l'intégrez dans le composant **Passthrough**.

Pour plus d'informations, voir [Métadonnées d'entête](https://docusaurus.io/fr/docs/markdown-features/head-metadata) dans la documentation de Docusaurus.
