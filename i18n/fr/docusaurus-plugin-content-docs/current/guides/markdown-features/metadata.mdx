# Métadonnées d'en-tête

DocStatic définit automatiquement pour vous des métadonnées utiles dans `<html>`, `<head>` et `<body>`.
Il est possible d'ajouter des métadonnées supplémentaires (ou de remplacer celles qui existent déjà) avec la balise `<head>` dans les fichiers Markdown.

Dans le CMS, celles-ci doivent être ajoutées dans l'éditeur Markdown. Elles ne seront pas reconnues si vous les intégrez dans le composant **Passthrough**.

Pour plus d'informations, consultez la section [Métadonnées d'en-tête](https://docusaurus.io/docs/markdown-features/head-metadata) dans la documentation Docusaurus.
