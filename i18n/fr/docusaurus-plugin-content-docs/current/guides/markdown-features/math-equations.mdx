# Équations mathématiques

Les équations mathématiques peuvent être rendues avec [KaTeX](https://katex.org).

## Utiliser

Pour utiliser KaTeX à partir du CMS, intégrez la chaîne dans un composant **Passthrough**. Vous pouvez inclure un **Summary** qui apparaîtra en ligne lors de l'édition du sujet.

### Incliné

Écrire des équations mathématiques en ligne en enveloppant les équations LaTeX entre `$` :

```latex
Let $f\colon[a,b]\to\R$ be Riemann integrable. Let $F\colon[a,b]\to\R$ be
$F(x)=\int_{a}^{x} f(t)\,dt$. Then $F$ is continuous, and at all $x$ such that
$f$ is continuous at $x$, $F$ is differentiable at $x$ with $F'(x)=f(x)$.
```

### Blocs

Pour le bloc d'équation ou le mode d'affichage, utilisez les sauts de ligne et `$$` :

```latex
$$
I = \int_0^{2\pi} \sin(x)\,dx
$$
```
