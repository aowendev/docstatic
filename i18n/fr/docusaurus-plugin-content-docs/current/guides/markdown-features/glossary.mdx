# Termes du glossaire

Le composant **Terme du glossaire** vous permet de saisir une clé qui correspond à un terme et à une description localisés. Le terme est affiché en souligné. Lorsque vous déplacez le curseur sur le terme, le curseur d'aide s'affiche et la description apparaît sous forme d'info-bulle. Sur les appareils à écran tactile, appuyez sur le terme pour afficher une définition contextuelle.

## Gestion des termes du glossaire

Vous pouvez modifier la collection de termes du glossaire dans le CMS.

1. Cliquez sur l'icône **Ajouter** ( **+** ) pour ajouter une nouvelle **Clé**.
2. Modifiez la clé, puis cliquez sur l'icône **Ajouter** ( **+** ) pour ajouter une nouvelle **Traduction**.
3. Modifiez la traduction.
4. Attribuez-lui un **Code de langue**.
5. Saisissez la manière dont le **Terme** doit apparaître dans le texte.
6. **Saisissez la Définition.**

Le terme du glossaire utilisera la version linguistique de la page sur laquelle il est affiché. Si aucune définition n'existe, il reviendra à la langue source.

## Utilisation des termes du glossaire

1. Sélectionnez **Terme du glossaire** dans la liste **Intégrer**.
2. Modifiez le composant.
3. Entrez la **Clé**.

#### Exemple

<GlossaryTerm termKey="single-sourcing" initcap={true} />
