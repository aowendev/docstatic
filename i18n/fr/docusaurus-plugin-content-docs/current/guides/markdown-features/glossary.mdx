# Glossaire

Le composant **Terme du glossaire** vous permet de saisir une clé qui correspond à un terme localisé et à une description. Le terme est souligné. Le passage du curseur sur le terme fait apparaître le curseur d'aide, et le survol affiche la description sous forme d'infobulle. Sur les appareils à écran tactile, le fait d'appuyer sur le terme affiche une définition dans une fenêtre contextuelle.

## Gestion des termes du glossaire

Vous pouvez modifier la collection de termes du glossaire dans le CMS.

1. Cliquez sur l'icône **Add** ( **+** ) pour ajouter une nouvelle **Key**.
2. Modifiez la clé, puis cliquez sur l'icône **Add** ( **+** ) pour ajouter une nouvelle **Translation**.
3. Modifiez la traduction.
4. Donnez-lui un **Language Code**.
5. Indiquez comment le **Term** doit apparaître dans le texte.
6. Saisissez la **Definition**.

Le terme du glossaire utilisera la version linguistique de la page sur laquelle il est affiché. S'il n'existe pas de définition, la langue source sera utilisée.

## Utiliser les termes du glossaire

1. Sélectionnez **Glossary Term** dans la liste **Embed**.
2. Modifiez le composant.
3. Saisissez la **Key**.

#### Exemple

<GlossaryTerm termKey="single-sourcing" />
