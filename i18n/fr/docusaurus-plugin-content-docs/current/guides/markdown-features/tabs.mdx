Le composant **Onglets** vous permet d'ajouter du contenu sous forme d'onglets à votre rubrique.

1. Sélectionnez **Onglets** dans la liste **Intégrer**.
2. Modifiez le composant.
3. Pour chaque onglet :
   1. Sélectionnez **Élément d'onglet** dans la liste **Intégrer**.
   2. Attribuez-lui une **Valeur**.
   3. Attribuez-lui un **Titre**.
   4. Saisissez le **Contenu**.
   5. (Facultatif) Définissez-le comme onglet **Par défaut**.
4. **Enregistrez** vos modifications.

<Admonition type="danger" title="Avertissement">
  Le CMS peut perdre le contexte lors de la modification des onglets. Vous devez enregistrer fréquemment. Le CMS peut se retrouver dans une situation où il ne parvient pas à trouver le contexte de l'onglet. Dans ce cas, cliquez sur l'icône **Raw Markdown** et apportez les modifications directement dans le Markdown. Enregistrez, puis revenez à l'éditeur de texte enrichi.
</Admonition>

#### Exemple :

<Tabs>
  <TabItem label="Apple" value="apple">
    Ceci est une pomme 🍎
  </TabItem>

  <TabItem label="Orange" value="orange">
    Ceci est une orange 🍊
  </TabItem>

  <TabItem label="Banana" value="banana">
    This is a banana 🍌
  </TabItem>
</Tabs>

Pour plus d'informations, consultez la section [Onglets](https://docusaurus.io/docs/markdown-features/tabs) de la documentation Docusaurus.
