# Chiffres

Le composant **Figures** vous permet d'ajouter des images avec des légendes à un sujet. Lorsqu'un utilisateur clique sur une figure, celle-ci s'affiche avec un agrandissement maximal dans une boîte lumineuse.

1. Sélectionnez **Figure** dans la liste **Embed**.
2. Modifiez le composant.
3. Sélectionnez une image dans le **Media Manager**.
4. Saisissez une légende. Celle-ci sera également utilisée comme élément de texte alternatif pour les lecteurs d'écran.
5. (Facultatif) Entrez une **Size**.
