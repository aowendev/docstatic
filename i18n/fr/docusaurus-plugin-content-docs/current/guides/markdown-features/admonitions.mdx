# Avertissements

Le composant **Avertissements** vous permet d'ajouter des avertissements stylisés.

1. Sélectionnez **Avertissement** dans la liste **Intégrer**.
2. Modifiez le composant et sélectionnez un **Type** (**Remarque**, **Conseil**, **Info**, **Attention** ou **Avertissement**).
3. Donnez-lui un **Titre**.
4. Saisissez le **Contenu**.

#### Exemples :

<Admonition type="note" title="Note">
  Utilisez les avertissements de type note pour fournir des informations supplémentaires.
</Admonition>

<Admonition type="tip" title="Conseil">
  Utilisez les avertissements de type conseil pour fournir des conseils que l'on peut ignorer sans risque.
</Admonition>

<Admonition type="info" title="Info">
  Utilisez les avertissements de type info pour les informations suffisamment importantes pour être mises en évidence par rapport au reste du texte.
</Admonition>

<Admonition type="caution" title="Mise en garde">
  Utilisez les avertissements de mise en garde lorsque quelque chose peut empêcher la réalisation d'une action prévue.
</Admonition>

<Admonition type="danger" title="Avertissement">
  Utilisez les avertissements d'avertissement lorsqu'il existe un risque de blessure, de faille de sécurité ou de perte de données.
</Admonition>
