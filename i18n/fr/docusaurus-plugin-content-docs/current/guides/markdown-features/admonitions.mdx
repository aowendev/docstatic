# Admonitions

Le composant **Admonitions** vous permet d'ajouter des avertissements stylisés.

1. Sélectionnez **Admonition** dans la liste **Embed**.
2. Modifiez le composant et sélectionnez un **Type** (**Note**, **Conseil**, **Info**, **Caution** ou **Warning**).
3. Donnez-lui un **Title**.
4. Saisissez le **Content**.

#### Exemples :

<Admonition type="note" title="Note">
  Utilisez les admonitions de note pour obtenir des informations supplémentaires.
</Admonition>

<Admonition type="tip" title="Conseil">
  Utilisez les avertissements de type "tip" pour fournir des conseils qu'il est prudent d'ignorer.
</Admonition>

<Admonition type="info" title="Info">
  Utilisez les avertissements de type "info" pour les informations suffisamment importantes pour se démarquer du reste du texte.
</Admonition>

<Admonition type="caution" title="Prudence">
  Utilisez les avertissements de prudence lorsque quelque chose risque d'empêcher la réalisation d'une action prévue.
</Admonition>

<Admonition type="danger" title="Avertissement">
  Utilisez des avertissements lorsqu'il existe un risque de blessure, de violation de la sécurité ou de perte de données.
</Admonition>
