# Ensembles de variables

Le composant Variable vous permet de placer une variable localisée dans votre texte.

## Gérer les jeux de variables

1. Dans le CMS, à partir de la collection Jeux de variables, ajoutez un nouveau jeu ou modifiez un jeu existant.
2. Saisissez ou modifiez le **Variable Set Name**.
3. Ajoutez ou modifiez une **Variable**.
4. Saisissez ou modifiez la **Key**.
5. Ajouter ou modifier une **Translation**.
6. Modifiez ou saisissez une paire **Language** et **Value**.

## Utiliser des variables

1. Sélectionnez le composant Variable dans la liste **Embed**.
2. Modifiez le composant.
3. Saisissez la clé **Variable Set**.
4. Saisissez la touche **Variable**.

#### Exemple

<VariableSet setKey="writing-terms" variableKey="single-sourcing" />
