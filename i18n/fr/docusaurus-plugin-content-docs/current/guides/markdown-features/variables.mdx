# Ensembles de variables

Le composant Variable vous permet d'insérer une variable localisée dans votre texte.

## Gérer les ensembles de variables

1. Dans le CMS, à partir de la collection **Ensembles de variables**, ajoutez un nouvel ensemble ou modifiez un ensemble existant.
2. Saisissez ou modifiez le **Nom de l'ensemble de variables**.
3. Ajoutez ou modifiez une **Variable**.
4. Entrez ou modifiez la **Clé**.
5. Ajoutez ou modifiez une **Traduction**.
6. Modifiez ou entrez une paire **Langue** et **Valeur**.

## Utiliser les variables

1. Sélectionnez le composant **Variable** dans la liste **Intégrer**.
2. Modifiez le composant.
3. Sélectionnez les clés **Variable** (**Ensemble de variables**).

#### Exemple

<VariableSet variableSelection="writing-terms|single-sourcing" initcap={true} />
