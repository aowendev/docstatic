# Internationalisation

La prise en charge de l'internationalisation ([i18n](https://fr.wikipedia.org/wiki/Internationalisation_et_localisation)) est assurée par le générateur de sites statiques. Pour des informations détaillées, voir [i18n - Introduction](https://docusaurus.io/fr/docs/i18n/introduction) dans la documentation de Docusuarus.

## Editer les documents traduits dans le CMS

1. Dans la collection **Translations**, sélectionnez le document que vous souhaitez éditer.
2. Effectuez vos modifications
3. Réglez le **Statut du document** sur **En traduction** ou **Traduction approuvée** selon le cas.
4. Cliquez sur **Sauvegarder**.
