# Internationalisation

La prise en charge de l'internationalisation ([i18n](https://en.wikipedia.org/wiki/Internationalization_and_localization)) est assurée par le générateur de site statique. Pour plus d'informations, consultez la section [i18n - Introduction](https://docusaurus.io/docs/i18n/introduction) de la documentation Docusuarus.

## Modifier les documents traduits dans le CMS

1. Dans la collection **Traductions**, sélectionnez le document que vous souhaitez modifier.
2. Apportez vos modifications.
3. Définissez le **Statut du document** sur **En cours de traduction** ou **Traduction approuvée** selon vos besoins.
4. Cliquez sur **Enregistrer**.
