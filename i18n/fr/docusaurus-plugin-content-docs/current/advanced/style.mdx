# Contrôle du style

L'une des fonctionnalités qui manque le plus aux rédacteurs travaillant dans des documents comme code est un bon correcteur d'orthographe, de grammaire et de style. Par défaut, le correcteur orthographique est fourni par votre navigateur web ou votre environnement de développement intégré. Cela peut suffire pour des sites web simples, mais si vous avez besoin de quelque chose de plus avancé, il vous faut un vérificateur de grammaire dédié.

## LanguageTool

DocStatic a été testé avec [LanguageTool](https://languagetool.org/fr/), un correcteur grammatical établi sur l'intelligence artificielle qui prend en charge plus de 20 langues. L'inscription à l'offre originelle est gratuite ([https://languagetool.org/fr/register](https://languagetool.org/fr/register)). Le [plan premium](https://languagetool.org/fr/premium) ajoute la prise en charge des guides de style pour les individus et les équipes. Des plans sur mesure pour les [entreprises](https://languagetool.org/fr/sales) sont disponibles. Il existe également une [réduction](https://languagetool.org/fr/premium) pour les étudiants et les organisations à but non lucratif.

## Auto-hébergé

LanguageTool étant un logiciel libre, vous pouvez de l'[auto-héberger](https://languagetool.org/fr/dev). Dans ce cas, aucuns frais n'existent d'abonnement. Vous aurez besoin d'une équipe de développement pour mettre en place votre instance de LanguageTool, et vous devrez assurer votre propre support.
