Avec l'iPadOS d'Apple, vous pouvez convertir n'importe quelle page web en une application web qui s'exécute dans sa propre instance de Safari avec sa propre icône sur votre écran d'accueil. Vous pouvez utiliser cette capacité pour créer une application pour le CMS hébergé dans le nuage.

1. Connectez-vous au CMS.
2. Naviguez jusqu'à la collection que vous souhaitez utiliser comme page d'accueil. Par exemple, la collection **Topics** ou la **Table of Contents**.
3. Appuyez sur l'icône **Partager**, appuyez sur **Plus** ( **…** ) et appuyez sur **Ajouter à l'écran d'accueil** ( **+** ).
4. Donnez un nom à l'application. Par défaut, elle utilisera TinaCMS.
5. Cliquez sur **Ajouter**.
