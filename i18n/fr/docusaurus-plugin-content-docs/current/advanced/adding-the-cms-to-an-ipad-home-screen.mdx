Avec Apple iPadOS, vous pouvez convertir n'importe quelle page Web en une application Web qui s'exécute dans sa propre instance de Safari avec sa propre icône sur votre écran d'accueil. Vous pouvez utiliser cette fonctionnalité pour créer une application pour le CMS hébergé dans le cloud.

1. Connectez-vous au CMS.
2. Accédez à la collection que vous souhaitez utiliser comme page d'accueil. Par exemple, la collection **Sujets** ou la **Table des matières**.
3. Appuyez sur l'icône **Partager**, puis sur **Plus** ( **…** ) et enfin sur **Ajouter à l'écran d'accueil** ( **+** ).
4. Donnez un nom à l'application. Par défaut, elle s'appellera TinaCMS.
5. Cliquez sur **Ajouter**.
