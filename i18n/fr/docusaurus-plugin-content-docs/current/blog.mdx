# Blog

La fonction blog permet de créer des articles horodatés.

<Admonition type="info" title="Infor">
  Reportez-vous à [Blog Plugin API Reference](https://docusaurus.io/docs/api/plugins/@docusaurus/plugin-content-blog) dans la documentation de Docusaurus pour une liste exhaustive des options.
</Admonition>

1. Dans la collection **Articles de blog** du CMS, cliquez sur **Ajouter un fichier**.
2. Donnez-lui un **Titre**.
3. Ajoutez des **Auteurs**. Pour chaque auteur, modifiez le composant et ajoutez :
   1. Leur **Nom**.
   2. Leur **Titre**.
   3. UNE **URL**.
   4. Une **URL d'image**.
4. Sélectionnez la **Date** de publication. Les articles postdatés n'apparaîtront pas tant que le site ne sera pas reconstruit.
5. (Facultatif) Ajoutez les **Tags** de la taxonomie.
6. Saisissez le contenu.
7. Cliquez sur **Sauvegarder**.

Pour plus d'informations, voir [Blog](https://docusaurus.io/fr/docs/blog) dans la documentation du Docusaurus.
