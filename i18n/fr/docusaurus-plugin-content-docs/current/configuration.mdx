# Configuration

La configuration du site DocStatic est gérée via le fichier `docusaurus.config.ts`. Pour plus d'informations sur la modification directe de ce fichier, consultez la section [Configuration](https://docusaurus.io/docs/configuration) de la documentation Docusaurus.

Pour utiliser le CMS afin de modifier la configuration, accédez à la collection **Paramètres**. À partir de là, vous pouvez configurer les éléments suivants du site :

* **Titre**.
* **Slogan**.
* **Favicon**.
* **URL du schéma GraphQL**.
* **Mode couleur** par défaut (foncé ou clair).
* Éléments du **pied de page**.
* **Langues**.
* **Logo**.
* Schémas **OpenAPI**.
* **Mise en évidence de la syntaxe**.
* **URL**.
* **Barre de navigation**.
* Paramètres de la **barre latérale**.
* Paramètres du **blog**.

Il existe une collection **Thème** distincte pour modifier les paramètres de couleur et de typographie.
