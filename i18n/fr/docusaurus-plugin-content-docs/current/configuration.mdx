# Configuration

La configuration du site DocStatic est gérée par le fichier `docusaurus.config.ts`. Pour plus d'informations sur l'édition directe de ce fichier, reportez-vous à [Configuration](https://docusaurus.io/fr/docs/configuration) dans la documentation de Docusaurus.

Pour utiliser le CMS afin de modifier la configuration, naviguez vers la collection [Settings](http://localhost:3000/admin#/collections/edit/settings/index). À partir de là, vous pouvez configurer le logo du site, le titre, l'accroche et le pied de page.
