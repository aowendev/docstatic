Le noyau du générateur de site statique ne fournit aucune fonctionnalité propre. Toutes les fonctionnalités sont déléguées à des plugins individuels : la fonctionnalité de documentation fournie par le [plugin docs](https://docusaurus.io/docs/api/plugins/@docusaurus/plugin-content-docs) ; la fonctionnalité blog fournie par le [plugin blog](https://docusaurus.io/docs/api/plugins/@docusaurus/plugin-content-docs) ; ou les pages individuelles fournies par le [plugin pages](https://docusaurus.io/docs/api/plugins/@docusaurus/plugin-content-docs).

DocStatic est préconfiguré pour utiliser les plugins nécessaires à la prise en charge :

* des équations mathématiques KaTeX
* de la recherche Lunr
* des diagrammes Mermaid
* des documents OpenAPI
* des documents GraphQL

Pour plus d'informations, consultez la section [Utilisation des plugins](https://docusaurus.io/docs/using-plugins) dans la documentation Docusaurus.
