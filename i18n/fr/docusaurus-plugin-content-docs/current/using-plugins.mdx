Le noyau du générateur de sites statiques ne fournit aucune fonctionnalité propre. Toutes les fonctionnalités sont déléguées à des plugins individuels : la fonctionnalité docs fournie par le [plugin docs](https://docusaurus.io/fr/docs/api/plugins/@docusaurus/plugin-content-docs) ; la fonctionnalité blog fournie par le [plugin blog](https://docusaurus.io/fr/docs/api/plugins/@docusaurus/plugin-content-docs) ; ou les pages individuelles fournies par le [plugin pages](https://docusaurus.io/fr/docs/api/plugins/@docusaurus/plugin-content-docs).

DocStatic est préconfiguré pour utiliser les plugins nécessaires au support :

* équations mathématiques KaTeX
* Recherche Lunr
* Diagrammes Mermaid
* Documents OpenAPI
* docs GraphQL

Pour plus d'informations, voir [Utilisation de Plugins](https://docusaurus.io/fr/docs/using-plugins)) dans la documentation de Docusaurus.
