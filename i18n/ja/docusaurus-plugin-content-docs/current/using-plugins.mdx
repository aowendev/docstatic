# プラグインの使用

静的サイトジェネレーターのコアは独自の機能を提供しません。すべての機能は個々のプラグインに委ねられています。ドキュメント機能は[docsプラグイン](https://docusaurus.io/docs/api/plugins/@docusaurus/plugin-content-docs)、ブログ機能は[blogプラグイン](https://docusaurus.io/docs/api/plugins/@docusaurus/plugin-content-blog)、個別ページは[pagesプラグイン](https://docusaurus.io/docs/api/plugins/@docusaurus/plugin-content-pages)によって提供されています。

docStaticは以下をサポートするために必要なプラグインが事前設定されています。

* KaTeX数式
* Lunr検索
* Mermaidダイアグラム
* OpenAPIドキュメント
* GraphQLドキュメント

詳細については、Docusaurusドキュメントの[Using Plugins](https://docusaurus.io/docs/using-plugins)を参照してください。
