# ドキュメント入門

ドキュメント機能は、Markdown ファイルを階層形式で整理する方法を提供します。

<Admonition type="info" title="情報">
  オプションの完全なリストについては、Docusaurus ドキュメントの [Docs Plugin API Reference](https://docusaurus.io/docs/api/plugins/@docusaurus/plugin-content-docs) を参照してください。
</Admonition>

ドキュメントのみのモードまたはブログのみのモードの設定については、Docusaurus ドキュメントの [Docs introduction](https://docusaurus.io/docs/docs-introduction) を参照してください。
