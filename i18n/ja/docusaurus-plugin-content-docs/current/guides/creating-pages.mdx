# ページの作成

ページは他の種類のコンテンツとは異なり、ショーケース、プレイグラウンド、サポートページなど、単独で完結するコンテンツとして設計されています。React コンポーネントと Markdown を含めることができます。ページにはサイドバーがありません。

<Admonition type="info" title="情報">
  オプションの完全なリストについては、Docusaurus ドキュメントの [Pages Plugin API Reference](http://localhost:3000/docs/api/plugins/plugin-content-pages.mdx) を参照してください。
</Admonition>

## ページを追加する

1. CMS の **Pages** コレクションから **Add File** をクリックします。
2. **Title** と **Description** を入力します。
3. リッチテキストエディタでコンテンツを追加します。
4. **Save** をクリックします。

## ページに React コンポーネントを追加する

docStatic の React コンポーネントはグローバルに利用可能です。ただし、1つのページでのみコンポーネントを使用する場合は、ページに直接コンポーネントをインポートすることもできます。ただし、CMS でページを編集できるようにするには、インポートディレクティブを [Passthrough](/docs/guides/markdown-features/react#passthrough-component) コンポーネントでラップする必要があります。docStatic サイトのサンプルページでは、ColorGenerator コンポーネントを次のようにインポートしています：

```markdown
<Passthrough type="jsx" string="import ColorGenerator from '@site/src/components/ColorGenerator';" summary="Import Statement" />

<Passthrough type="jsx" string="<ColorGenerator />" summary="JSX Component" />
```

詳細については、Docusaurus ドキュメントの [Creating Pages](https://docusaurus.io/docs/creating-pages) を参照してください。
