# Wikis

ウィキページは、コラボレーションのために設計されたウェブ/ドキュメントページです。数分で作成・編集でき、チーム内部のナレッジ共有や公開向けヘルプコンテンツによく使用されます。

CMSでウィキトピックを作成する場合：

* `/docs/wiki/` フォルダーにウィキトピックを作成します。
* ウィキトピックを目次に追加する必要はありません。ウィキトピックのサイドバーは、リポジトリのファイル構造に基づいて自動的に生成されます。
* ウィキトピックにはメタデータは含まれません。

docStaticはカテゴリをそのインデックスドキュメントに自動的にリンクできます。カテゴリインデックスドキュメントは、次のファイル名規則に従うドキュメントです。

`README`（大文字・小文字を問わず）という名前：`docs/wiki/folder/readme.mdx`

詳細については、Docusaurusドキュメントの[Autogenerated](https://docusaurus.io/docs/sidebar/autogenerated#autogenerated-sidebar-metadata)を参照してください。

## CMSでウィキページを作成する

1. **Wiki Pages** コレクションを選択します。
2. **Add File** をクリックします。

   <Figure img="/img/docs/wiki-pages-add-wiki.png" caption="TinaCMS。WikiオプションとAdd fileオプションがコールアウトでハイライトされています。" />
3. 新しいウィキページに **タイトル** を付けます。
4. コンテンツを入力します。
5. **Save** をクリックします。
