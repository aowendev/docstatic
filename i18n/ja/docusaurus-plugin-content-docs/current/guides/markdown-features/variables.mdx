# Variable sets

Variableコンポーネントを使用すると、テキストにローカライズされた変数を配置できます。

## 変数セットの管理

1. CMSの **Variable Sets** コレクションから、新しいセットを追加するか、既存のセットを編集します。
2. **変数セット名** を入力または編集します。
3. **変数** を追加または編集します。
4. **キー** を入力または編集します。
5. **翻訳** を追加または編集します。
6. **言語** と **値** のペアを編集または入力します。

## 変数の使用

1. **埋め込み** リストから **Variable** コンポーネントを選択します。
2. コンポーネントを編集します。
3. **変数**（**Variable Set**）のキーを選択します。

#### 例

<VariableSet variableSelection="writing-terms_single-sourcing" initcap={true} />
