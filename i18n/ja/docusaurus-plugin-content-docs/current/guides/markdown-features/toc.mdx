# Headings and table of contents

CMSでは **段落** または **見出し** レベル（1〜6）を選択できます。

<a id="anchor-example" />

## 見出しID

見出しIDは自動的に生成されます。たとえば、`heading-ids` のように生成されます。これらを使用して、トピックの見出しに直接リンクできます。ただし、見出しのタイトルを変更すると、リンクが壊れます。このため、docStaticは **Context Help** コンポーネントを提供しています。これにより、見出しだけでなくページ上のどこにでもアンカータグを埋め込むことができます。

1. **埋め込み** リストから **Context Help** を選択します。
2. コンポーネントを編集して **Context Help ID** を設定します。

アンカーにリンクするには、パスを使用します：`<URL>/docs/<slug>#<ID>.`

例：[https://docstatic.com/docs/guides/markdown-features/toc#anchor-example](https://docstatic.com/docs/guides/markdown-features/toc#anchor-example)

## 目次の見出しレベル

各Markdownドキュメントは右上に目次を表示します。デフォルトでは、この目次は `h2` と `h3` の見出しのみを表示します。これはページ構造の概要として十分なはずです。表示する見出しの範囲を変更する必要がある場合は、ページごとまたはグローバルに最小・最大見出しレベルをカスタマイズできます。

詳細については、Docusaurusドキュメントの[Headings and Table of contents](https://docusaurus.io/docs/markdown-features/toc)を参照してください。
