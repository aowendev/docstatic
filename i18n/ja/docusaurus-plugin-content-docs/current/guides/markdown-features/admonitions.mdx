# 注意書き

**Admonitions** コンポーネントを使用すると、スタイル付きの注意書きを追加できます。

1. **Embed** リストから **Admonition** を選択します。
2. コンポーネントを編集して **Type**（**Note**、**Tip**、**Info**、**Caution**、**Warning**）を選択します。
3. **Title** を付けます。
4. **Content** を入力します。

#### 例：

<Admonition type="note" title="注意">
  補足情報には Note（注意）の注意書きを使用します。
</Admonition>

<Admonition type="tip" title="ヒント">
  無視しても問題ないアドバイスを提供するには Tip（ヒント）の注意書きを使用します。
</Admonition>

<Admonition type="info" title="情報">
  本文から切り離して強調する必要があるほど重要な情報には Info（情報）の注意書きを使用します。
</Admonition>

<Admonition type="caution" title="注意">
  意図したアクションの完了を妨げる可能性があるものには Caution（注意）の注意書きを使用します。
</Admonition>

<Admonition type="danger" title="警告">
  負傷、セキュリティ侵害、またはデータ損失のリスクがある場合は Warning（警告）の注意書きを使用します。
</Admonition>
