# タブ

**Tabs** コンポーネントを使用すると、トピックにタブ付きコンテンツを追加できます。

1. **埋め込み** リストから **Tabs** を選択します。
2. コンポーネントを編集します。
3. 各タブについて：
   1. **埋め込み** リストから **Tab Item** を選択します。
   2. **値（Value）** を設定します。
   3. **タイトル（Title）** を設定します。
   4. **コンテンツ** を入力します。
   5. （オプション）**デフォルト** タブに設定します。
4. 変更を **保存** します。

<Admonition type="danger" title="警告">
  CMSはタブの編集中にコンテキストを失うことがあります。こまめに保存してください。CMSがタブのコンテキストを見つけられない状態になることがあります。その場合は、**Raw Markdown** アイコンをクリックして、Markdownを直接編集してください。保存してからリッチテキストエディターに戻ってください。
</Admonition>

#### 例:

<Tabs>
  <TabItem label="Apple" value="apple">
    This is an apple 🍎
  </TabItem>

  <TabItem label="Orange" value="orange">
    This is an orange 🍊
  </TabItem>

  <TabItem label="Banana" value="banana">
    This is a banana 🍌
  </TabItem>
</Tabs>

詳細については、Docusaurusドキュメントの[Tabs](https://docusaurus.io/docs/markdown-features/tabs)を参照してください。
