# Markdown links

別のページへのリンクを追加する方法は2つあります。URLパスを使用する方法とファイルパスを使用する方法です。

```
- [URL path to another document](./installation)
- [file path to another document](./installation.mdx)
```

CMSでは、テキストの一部を選択してペーストすることで、クリップボードからリンクを追加できます。

また、テキストの一部をハイライトして、リンクアイコンをクリックしてパスを追加することもできます。

または、リンクを表示したい場所にカーソルを置き、アイコンをクリックしてパスと表示テキストを入力することもできます。

リンクの詳細については、Docusaurusドキュメントの[Markdownリンク](https://docusaurus.io/docs/markdown-features/links)を参照してください。
