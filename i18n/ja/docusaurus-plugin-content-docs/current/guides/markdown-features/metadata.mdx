# Head metadata

docStaticは、`<html>`、`<head>`、`<body>` に有用なページメタデータを自動的に設定します。Markdownファイルの `<head>` タグを使用して、追加のメタデータを追加したり、既存のものを上書きしたりすることができます。

CMSでは、これはMarkdownエディターで追加する必要があります。**Passthrough** コンポーネントでラップしても認識されません。

詳細については、Docusaurusドキュメントの[Head metadata](https://docusaurus.io/docs/markdown-features/head-metadata)を参照してください。
