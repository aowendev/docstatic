# MDX plugins

MDXには、MarkdownファイルをJSXに解析・変換する方法をカスタマイズするために使用できる組み込みの[プラグインシステム](https://mdxjs.com/advanced/plugins/)があります。ただし、Markdownの動作を変更する場合は、CMSにもその変更を反映する必要があります。

<Admonition type="danger" title="警告">
  標準的なMarkdownコンポーネントの動作を変更するMDXプラグインは、CMSとの互換性を損なう可能性があります。自己責任で進めてください。
</Admonition>

詳細については、Docusaurusドキュメントの[MDX Plugins](https://docusaurus.io/docs/markdown-features/plugins)を参照してください。

<Admonition type="note" title="注意">
  docStaticは[数式プラグイン](math-equations)を事前にインストールして設定します。
</Admonition>
