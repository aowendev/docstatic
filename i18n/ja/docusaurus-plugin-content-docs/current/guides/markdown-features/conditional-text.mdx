# 条件付きテキスト

**Conditional Text** コンポーネントを使用すると、コンテンツをいつ表示するかを決定する条件のセットでラップできます：

* 条件には、言語またはユーザー定義条件の任意の組み合わせを指定できます。
* 1つ以上の条件が満たされた場合（OR）、またはすべての条件が満たされた場合のみ（AND）、コンテンツを表示または非表示にできます。
  条件が満たされない場合に表示するフォールバックテキストも指定できます。

## カテゴリと条件を作成する

1. CMS で **Conditions** コレクションをクリックします。
2. **Add** アイコンをクリックして新しい **Category**（カテゴリ）を追加します。
3. **Name**（名前）と **Description**（説明）を入力します。
4. 追加する各 **Condition**（条件）について：
   1. **Add** アイコンをクリックします。
   2. **Condition** に名前を付けて **Description** を入力します。
   3. 必要に応じて条件の **Active**（アクティブ）状態を切り替えます。

## トピックに条件を追加する

条件はトピックレベルで設定されます。言語はページの言語によって決まります。その他の条件は、トピックの先頭のメタデータに配列として定義されます。

1. トピックを作成または編集します。
2. **Conditions** エリアで **Categories** を展開し、トピックに適用したい **Conditions** のボックスを選択します。

## 条件付きテキストを追加する

1. **Embed** リストから **Conditional Text** を選択します。
2. コンポーネントを編集します。
3. **Content**（コンテンツ）を入力します。
4. **Action**（アクション）を設定します（表示または非表示）。
5. **Required Conditions**（必要な条件）を追加します。
6. **Required Languages**（必要な言語）を追加します。
7. **Condition Logic**（条件のロジック）を設定します。
8. **Language Logic**（言語のロジック）を設定します。
9. 条件と言語の両方が必要かどうかを選択します。
10. （オプション）**Fallback Text**（フォールバックテキスト）を入力します。

### 例

<ConditionalText action="show" conditions={["production"]} languages={[]} logic="any" languageLogic="any" requireBothConditions={false} fallback="">The `production` condition is set on this page.</ConditionalText>
