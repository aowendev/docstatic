# Glossary terms

**Glossary Term** コンポーネントを使用すると、ローカライズされた用語と説明にマッピングされるキーを入力できます。用語は下線付きで表示されます。用語の上にカーソルを移動するとヘルプカーソルが表示され、ホバーすると説明がツールチップとして表示されます。タッチスクリーンデバイスでは、用語をタップするとポップアップ定義が表示されます。

## 用語集の管理

用語集の用語はCMSのGlossary Termsコレクションで編集できます。

1. **追加** アイコン（ **+** ）をクリックして新しい **キー** を追加します。
2. キーを編集し、**追加** アイコン（ **+** ）をクリックして新しい **翻訳** を追加します。
3. 翻訳を編集します。
4. **言語コード** を設定します。
5. **用語** がテキスト内でどのように表示されるかを入力します。
6. **定義を入力します。**

Glossary Termは、表示されているページの言語バージョンを使用します。定義が存在しない場合は、ソース言語にフォールバックします。

## 用語集の使用

1. **埋め込み** リストから **Glossary Term** を選択します。
2. コンポーネントを編集します。
3. **キー** を入力します。

#### 例

<GlossaryTerm termKey="single-sourcing" initcap={true} />
