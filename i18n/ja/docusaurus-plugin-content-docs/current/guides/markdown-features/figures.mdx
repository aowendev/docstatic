# 図

Figures コンポーネントを使用すると、キャプション付きの画像をトピックに追加できます。ユーザーが図をクリックすると、ライトボックス内で最大倍率で表示されます。

1. Embed リストから Figure を選択します。
2. コンポーネントを編集します。
3. Media Manager から画像を選択します。
4. キャプションを入力します。これはスクリーンリーダー用の alt テキスト要素としても使用されます。
5. （オプション）サイズを入力します。
