# Figures

The Figures component enables you add images with captions to a topic. When a user clicks a figure, it is displayed at maximum magnification inside a lightbox.

1. Select Figure from the Embed list.
2. Edit the component.
3. Select an image from the Media Manager.
4. Enter a Caption. This will also be used as the alt-text element for screen readers.
5. (Optional) Enter a size.
