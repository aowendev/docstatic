# コメント

**Comment** コンポーネントを使用すると、CMS では表示されるが、生のソースコードを含めレンダリングされたページには表示されないコメントを追加できます。

1. **Embed** リストから **Comment** を選択します。
2. コンポーネントを編集します。
3. **Reviewer**（レビュアー）を入力します。
4. **Comment**（コメント）を入力します。

#### 例

<Comment comment="This comment is invisible in the published website." reviewer="AOwenDev" />
