# ブログ

ブログ機能を使用すると、タイムスタンプ付きの記事を作成できます。

<Admonition type="info" title="情報">
  オプションの完全なリストについては、Docusaurus ドキュメントの [Blog Plugin API Reference](https://docusaurus.io/docs/api/plugins/@docusaurus/plugin-content-blog) を参照してください。
</Admonition>

1. CMS の **Blog Articles** コレクションから **Add File** をクリックします。
2. **Title** を入力します。
3. **Authors** を追加します。各著者に対して、コンポーネントを編集して以下を追加します：
   1. 著者の **Name**。
   2. 著者の **Title**（肩書き）。
   3. **URL**。
   4. **Image URL**（画像 URL）。
4. 公開 **Date** を選択します。本番環境では、未来日付の記事はサイトが再ビルドされるまで表示されません。ただし、開発サーバー（`npm run dev`）の実行中は表示されます。
5. （オプション）タクソノミーの **Tags** を追加します。
6. コンテンツを入力します。
7. **Save** をクリックします。

詳細については、Docusaurus ドキュメントの [Blog](https://docusaurus.io/docs/blog) を参照してください。
