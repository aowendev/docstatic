docStatic では、[browserslist 設定](https://github.com/browserslist/browserslist)を使用してサポートするブラウザのリストを定義できます。通常、ウェブサイトは最新のブラウザで問題なく動作します。

詳細については、Docusaurus ドキュメントの [Browser support](https://docusaurus.io/docs/browser-support) を参照してください。

## CMS での編集

TinaCMS は以下のブラウザで完全にサポートされています：

* Chrome
* Edge
* Safari

<Admonition type="danger" title="警告">
  TinaCMS は Firefox に既知の問題があり、文字スタイルが無視されるものから作業が保存されないものまで、さまざまな問題が発生する可能性があります。別のブラウザを使用してください。
</Admonition>
