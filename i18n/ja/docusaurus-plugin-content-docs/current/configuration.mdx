# 設定

docStatic のサイト設定は `docusaurus.config.ts` ファイルで管理されます。このファイルを直接編集する方法については、Docusaurus ドキュメントの [Configuration](https://docusaurus.io/docs/configuration) を参照してください。

CMS を使用して設定を編集するには、**Settings** コレクションに移動します。ここから、サイトの以下の設定を変更できます：

* **Title**（タイトル）。
* **Tagline**（タグライン）。
* **Favicon**（ファビコン）。
* **GraphQL Schema URL**（GraphQL スキーマ URL）。
* デフォルトの **Colour Mode**（カラーモード）（ダークまたはライト）。
* **Footer**（フッター）アイテム。
* **Languages**（言語）。
* **Logo**（ロゴ）。
* **OpenAPI** スキーマ。
* **Syntax Highlighting**（シンタックスハイライト）。
* **URL**。
* **Navbar**（ナビゲーションバー）。
* **Sidebar**（サイドバー）設定。
* **Blog**（ブログ）設定。

カラースキームとタイポグラフィ設定を変更するための **Theme** コレクションが別途用意されています。
