docStatic supports search engine optimisation in various ways:

* Global metadata
* Human readable links

* Rich search information
* Robots file
* Single page metadata
* Sitemap file
* Structured content

For more information, refer to the [Search engine optimisation](https://docusaurus.io/docs/seo) in the Docusaurus documentation.
