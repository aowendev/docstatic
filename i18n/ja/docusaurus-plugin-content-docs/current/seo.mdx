docStaticは以下のさまざまな方法で検索エンジン最適化をサポートしています。

* グローバルメタデータ
* 人間が読みやすいリンク
* 画像のメタ説明
* リッチ検索情報
* Robotsファイル
* 単一ページのメタデータ
* サイトマップファイル
* 構造化コンテンツ

詳細については、Docusaurusドキュメントの[Search engine optimisation](https://docusaurus.io/docs/seo)を参照してください。
