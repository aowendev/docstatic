# Internationalization

国際化（[i18n](https://en.wikipedia.org/wiki/Internationalization_and_localization)）のサポートは静的サイトジェネレーターによって提供されています。詳細については、Docusaurusドキュメントの[i18n - Introduction](https://docusaurus.io/docs/i18n/introduction)を参照してください。

## CMSで翻訳済みドキュメントを編集する

1. **Translations** コレクションから、編集したいドキュメントを選択します。
2. 変更を加えます。
3. 必要に応じて **ドキュメントステータス** を **翻訳中** または **翻訳承認済み** に設定します。
4. **Save** をクリックします。
