# スタイルチェック

ドキュメント・アズ・コードで作業するライターが最も恋しく思う機能の1つが、優れたスペル・文法・スタイルチェッカーです。デフォルトでは、スペルチェックはウェブブラウザや統合開発環境によって提供されます。シンプルなウェブサイトには十分かもしれませんが、より高度なものが必要な場合は専用の文法チェッカーが必要です。

## LanguageTool

docStatic は [LanguageTool](https://languagetool.org/) でテストされています。これは20以上の言語をサポートする AI ベースの文法チェッカーです。基本プランへの[登録](https://languagetool.org/register)は無料です。[プレミアムプラン](https://languagetool.org/premium)は個人やチーム向けのスタイルガイドサポートを追加します。カスタマイズされた[ビジネス](https://languagetool.org/sales)プランも利用可能です。学生や非営利団体向けの[割引](https://languagetool.org/premium)もあります。

## セルフホスト

LanguageTool はオープンソースであるため、[セルフホスティング](https://languagetool.org/dev)という選択肢もあります。この場合、サブスクリプション料金は発生しませんが、LanguageTool インスタンスのセットアップには開発チームが必要で、サポートは自前で提供する必要があります。
