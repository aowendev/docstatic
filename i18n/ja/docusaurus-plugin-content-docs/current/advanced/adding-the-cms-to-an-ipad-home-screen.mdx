# CMSをiPadのホーム画面に追加する

Apple iPadOS では、任意のウェブページを独自の Safari インスタンスで動作するウェブアプリに変換し、ホーム画面に独自のアイコンとして追加できます。この機能を利用して、クラウドホスト型 CMS 用のアプリを作成できます。

1. CMS にサインインします。
2. ホームページとして使用したいコレクションに移動します。たとえば、**Topics** コレクションや **Table of Contents** などです。
3. **Share** アイコンをタップし、**More**（**…**）をタップして、**Add to Home Screen**（**+**）をタップします。
4. アプリに名前を付けます。デフォルトでは TinaCMS が使用されます。
5. **Add** をクリックします。
