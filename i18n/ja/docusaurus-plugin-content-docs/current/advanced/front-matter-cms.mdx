# Front Matter CMS

[VSCodium](https://vscodium.com/) などの IDE でコンテンツを作成する場合、CMS が提供する機能は利用できません。一般的なワークフローでは、開発者が IDE で MDX ファイルを直接編集してコンテンツを作成・編集し、ライターが CMS でコンテンツを整理してタグ付けします。開発者が最終更新日の記載を忘れた場合、その情報を取得するにはファイルのプロパティを確認する必要があります。ただし、[Front Matter](https://frontmatter.codes/) というプラグインとして利用できる軽量 CMS があります。docStatic リポジトリには Front Matter の設定例が含まれています。詳細については、Front Matter の[ドキュメント](https://frontmatter.codes/docs)を参照してください。
