# Upgrading

docStaticはまだ初期開発段階にあるため、変更はメインリポジトリから個別にマージする必要があります。ただし、メインリポジトリの最新バージョンで[package.json](https://github.com/aowendev/docstatic/blob/main/package.json)ファイルを置き換えることで、Node.jsコンポーネントを安全に更新できます。更新後は次を実行します。

```shell
yarn install
```
