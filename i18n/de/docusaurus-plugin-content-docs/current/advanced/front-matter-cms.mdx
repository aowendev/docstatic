Beim Erstellen von Inhalten in einer IDE wie [VSCodium](https://vscodium.com/) stehen die Funktionen des CMS nicht zur Verfügung. In einem typischen Workflow erstellen oder bearbeiten Entwickler Inhalte, indem sie die MDX-Datei direkt in der IDE ändern, während Autoren die Inhalte im CMS bereinigen und mit Tags versehen. Wenn der Entwickler vergisst, ein Datum für die letzte Änderung anzugeben, müssen Sie die Dateieigenschaften überprüfen, um diese Information zu erhalten. Es gibt jedoch ein leichtgewichtiges CMS, das als Plugin namens [Front Matter](https://frontmatter.codes/) verfügbar ist. Das DocStatic-Repository enthält ein Beispiel für eine Front Matter-Konfiguration. Weitere Informationen finden Sie in der Front Matter-[Dokumentation](https://frontmatter.codes/docs).
