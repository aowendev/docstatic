Mit Apple iPadOS können Sie jede Webseite in eine Web-App umwandeln, die in einer eigenen Instanz von Safari mit einem eigenen Symbol auf Ihrem Startbildschirm ausgeführt wird. Mit dieser Funktion können Sie eine App für das cloudbasierte CMS erstellen.

1. Melden Sie sich beim CMS an.
2. Navigieren Sie zu der Sammlung, die Sie als Startseite verwenden möchten. Zum Beispiel die Sammlung **Themen** oder das **Inhaltsverzeichnis**.
3. Tippen Sie auf das Symbol **Teilen**, tippen Sie auf **Mehr** ( **…** ) und tippen Sie auf **Zum Startbildschirm hinzufügen** ( **+** ).
4. Geben Sie der App einen Namen. Standardmäßig wird TinaCMS verwendet.
5. Klicken Sie auf **Hinzufügen**.
