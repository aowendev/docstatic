Standardmäßig ist DocStatic für die Verwendung der [Lunr](https://lunrjs.com/)-Suche konfiguriert. Der Hauptvorteil dieser Lösung besteht darin, dass sie kostenlos ist. Der größte Nachteil ist, dass der gesamte Suchindex auf den Browser des Benutzers heruntergeladen wird. Je größer Ihre Website ist, desto langsamer wird sie.

Als Docusaurus-Plugins stehen verschiedene Alternativen zur Verfügung, darunter:

* Algolia (vorinstalliert)
* [Markprompt](https://github.com/typesense/docusaurus-theme-search-typesense)
* [Meilisearch](https://github.com/tauri-apps/meilisearch-docsearch)
* [Orama](https://github.com/askorama/orama/tree/main/packages/plugin-docusaurus-v3)
* [Typesense DocSearch](https://github.com/typesense/docusaurus-theme-search-typesense)

Weitere Informationen finden Sie unter [Suche](https://docusaurus.io/docs/search#using-algolia-docsearch) in der Docusaurus-Dokumentation.
