# Blog

Mit der Blog-Funktion können Sie Artikel mit Zeitstempel erstellen.

<Admonition type="info" title="Infor">
  Eine vollständige Liste der Optionen finden Sie in der Docusaurus-Dokumentation unter [Blog Plugin API Reference](https://docusaurus.io/docs/api/plugins/@docusaurus/plugin-content-blog).
</Admonition>

1. Klicken Sie in der CMS-Sammlung **Blog-Artikel** auf **Datei hinzufügen**.
2. Geben Sie einen **Titel** ein.
3. Fügen Sie **Autoren** hinzu. Bearbeiten Sie für jeden Autor die Komponente und fügen Sie Folgendes hinzu:
   1. Den **Namen**.
   2. Den **Titel**.
   3. Eine **URL**.
   4. Eine **Bild-URL**.
4. Wählen Sie das **Datum** der Veröffentlichung aus. Artikel mit einem späteren Datum werden erst angezeigt, wenn die Website neu erstellt wurde.
5. (Optional) Fügen Sie beliebige Taxonomie-**Tags** hinzu.
6. Geben Sie den Inhalt ein.
7. Klicken Sie auf **Speichern**.

Weitere Informationen finden Sie unter [Blog](https://docusaurus.io/docs/blog) in der Docusaurus-Dokumentation.
