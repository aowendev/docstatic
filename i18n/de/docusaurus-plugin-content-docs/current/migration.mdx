# Aktualisierung

Da sich DocStatic noch in einer frühen Entwicklungsphase befindet, müssen Änderungen einzeln aus dem Haupt-Repository übernommen werden. Die Node.js-Komponenten können jedoch sicher aktualisiert werden, indem die Datei [package.json](https://github.com/aowendev/docstatic/blob/main/package.json) durch die neueste Version aus dem Haupt-Repository ersetzt wird. Führen Sie nach der Aktualisierung folgenden Befehl aus:

```shell
yarn install
```
