Mit DocStatic können Sie die Liste der unterstützten Browser über eine [Browsersliste-Konfiguration](https://github.com/browserslist/browserslist) definieren. In der Regel sollten Websites auf jedem modernen Browser gut funktionieren.

Weitere Informationen finden Sie unter [Browserunterstützung](https://docusaurus.io/docs/browser-support) in der Docusaurus-Dokumentation.

## Bearbeiten im CMS

TinaCMS wird vollständig unterstützt von:

* Chrome
* Edge
* Safari

<Admonition type="danger" title="Warnung">
  TinaCMS hat bekannte Probleme mit Firefox, die von ignorierten Zeichenformaten bis hin zu nicht gespeicherten Änderungen reichen können. Sie sollten einen anderen Browser verwenden.
</Admonition>
