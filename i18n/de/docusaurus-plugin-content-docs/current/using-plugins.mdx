Der Kern des statischen Website-Generators bietet keine eigenen Funktionen. Alle Funktionen werden an einzelne Plugins delegiert: die Dokumentationsfunktion wird vom [Dokumentations-Plugin](https://docusaurus.io/docs/api/plugins/@docusaurus/plugin-content-docs) bereitgestellt; die Blog-Funktion wird vom [Blog-Plugin](https://docusaurus.io/docs/api/plugins/@docusaurus/plugin-content-docs) bereitgestellt; oder einzelne Seiten werden vom [Seiten-Plugin](https://docusaurus.io/docs/api/plugins/@docusaurus/plugin-content-docs) bereitgestellt.
