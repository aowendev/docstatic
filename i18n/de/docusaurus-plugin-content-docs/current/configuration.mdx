# Konfiguration

Die Konfiguration der DocStatic-Website wird über die Datei `docusaurus.config.ts` verwaltet. Informationen zum direkten Bearbeiten dieser Datei finden Sie unter [Konfiguration](https://docusaurus.io/docs/configuration) in der Docusaurus-Dokumentation.

Um die Konfiguration mit dem CMS zu bearbeiten, navigieren Sie zur Sammlung „Einstellungen“. Von hier aus können Sie Folgendes für die Website konfigurieren:

* **Titel**.
* **Slogan**.
* **Favicon**.
* **GraphQL-Schema-URL**.
* Standardmäßiger **Farbmodus** (dunkel oder hell).
* **Fußzeilenelemente**.
* **Sprachen**.
* **Logo**.
* **OpenAPI**-Schemas.
* **Syntaxhervorhebung**.
* **URL**.
* **Navigationsleiste**.
* **Seitenleiste**-Einstellungen.
* **Blog**-Einstellungen.

Es gibt eine separate **Theme**-Sammlung zum Ändern der Farbschema- und Typografie-Einstellungen.
