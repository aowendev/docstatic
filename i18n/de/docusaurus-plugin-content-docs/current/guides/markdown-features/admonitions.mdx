# Hinweise

Mit der Komponente **Hinweise** können Sie formatierte Hinweise hinzufügen.

1. Wählen Sie **Hinweis** aus der Liste **Einbetten** aus.
2. Bearbeiten Sie die Komponente und wählen Sie einen **Typ** aus (**Hinweis**, **Tipp**, **Info**, **Vorsicht** oder **Warnung**).
3. Geben Sie einen **Titel** ein.
4. Geben Sie den **Inhalt** ein.

#### Beispiele:

<Admonition type="note" title="Hinweis">
  Verwenden Sie Hinweis-Admonitions für ergänzende Informationen.
</Admonition>

<Admonition type="tip" title="Tip">
  Verwenden Sie Hinweis-Admonitions, um Ratschläge zu geben, die ignoriert werden können. 
</Admonition>

<Admonition type="info" title="Info">
  Verwenden Sie Info-Admonitions für Informationen, die so wichtig sind, dass sie sich vom Rest des Textes abheben müssen. 
</Admonition>

<Admonition type="caution" title="Vorsicht">
  Verwenden Sie Vorsichtshinweise, wenn etwas die Ausführung einer beabsichtigten Aktion verhindern könnte. 
</Admonition>

<Admonition type="danger" title="Warnung">
  Verwenden Sie Warnhinweise, wenn die Gefahr von Verletzungen, Sicherheitsverletzungen oder Datenverlusten besteht. 
</Admonition>
