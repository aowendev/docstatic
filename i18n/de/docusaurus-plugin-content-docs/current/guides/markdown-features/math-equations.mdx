# Mathematische Gleichungen

Mathematische Gleichungen können mit [KaTeX](https://katex.org) dargestellt werden.

## Verwendung

Um KaTeX aus dem CMS zu verwenden, binden Sie die Zeichenfolge in eine **Passthrough**-Komponente ein. Sie können eine **Zusammenfassung** hinzufügen, die beim Bearbeiten des Themas inline angezeigt wird.

### Inline

Schreiben Sie Inline-Mathematikgleichungen, indem Sie LaTeX-Gleichungen zwischen `$` einschließen:

```latex
Let $f\colon[a,b]\to\R$ be Riemann integrable. Let $F\colon[a,b]\to\R$ be
$F(x)=\int_{a}^{x} f(t)\,dt$. Then $F$ is continuous, and at all $x$ such that
$f$ is continuous at $x$, $F$ is differentiable at $x$ with $F'(x)=f(x)$.
```

### Blöcke

Für Gleichungsblöcke oder den Anzeigemodus verwenden Sie Zeilenumbrüche und `$$`:

```latex
$$
I = \int_0^{2\pi} \sin(x)\,dx
$$
```
