# Markdown-Links

Es gibt zwei Möglichkeiten, einen Link zu einer anderen Seite hinzuzufügen: über einen URL-Pfad oder einen Dateipfad.

```
- [URL-Pfad zu einem anderen Dokument](./installation)
- [Dateipfad zu einem anderen Dokument](./installation.mdx)
```

Im CMS können Sie einen Link aus der Zwischenablage hinzufügen, indem Sie einen Textabschnitt auswählen und einfügen.

Sie können auch einen Textabschnitt markieren und auf das Symbol „Link“ klicken, um einen Pfad hinzuzufügen.

Oder Sie können den Cursor an die Stelle setzen, an der der Link erscheinen soll, auf das Symbol klicken und den Pfad und den Anzeigetext eingeben.

Weitere Informationen zu Links finden Sie unter [Markdown-Links](https://docusaurus.io/docs/markdown-features/links) in der Docusuarus-Dokumentation.
