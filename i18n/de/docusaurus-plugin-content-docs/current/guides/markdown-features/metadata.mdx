# Kopf-Metadaten

DocStatic setzt automatisch nützliche Seiten-Metadaten in `<html>`, `<head>` und `<body>` für Sie.
Es ist möglich, zusätzliche Metadaten hinzuzufügen (oder vorhandene zu überschreiben) mit dem `<head>`-Tag in Markdown-Dateien.

Im CMS muss dies im Markdown-Editor hinzugefügt werden. Es wird nicht erkannt, wenn Sie es in die **Passthrough**-Komponente einfügen.

Weitere Informationen finden Sie unter [Head-Metadaten](https://docusaurus.io/docs/markdown-features/head-metadata) in der Docusaurus-Dokumentation.
