# Variablensätze

Mit der Komponente „Variable“ können Sie eine lokalisierte Variable in Ihren Text einfügen.

## Variablensätze verwalten

1. Fügen Sie im CMS aus der Sammlung **Variablensätze** einen neuen Satz hinzu oder bearbeiten Sie einen vorhandenen Satz.
2. Geben Sie den **Namen des Variablensatzes** ein oder bearbeiten Sie ihn.
3. Fügen Sie eine **Variable** hinzu oder bearbeiten Sie sie.
4. Geben Sie den **Schlüssel** ein oder bearbeiten Sie ihn.
5. Fügen Sie eine **Übersetzung** hinzu oder bearbeiten Sie sie.
6. Bearbeiten Sie ein Paar aus **Sprache** und **Wert** oder geben Sie es ein.

## Variablen verwenden

1. Wählen Sie die Komponente **Variable** aus der Liste **Einbetten** aus.
2. Bearbeiten Sie die Komponente.
3. Wählen Sie die Schlüssel der **Variablen** (**Variablensatz**) aus.

#### Beispiel

<VariableSet variableSelection="writing-terms|single-sourcing" initcap={true} />
