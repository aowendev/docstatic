# Glossarbegriffe

Mit der Komponente **Glossarbegriff** können Sie einen Schlüssel eingeben, der einem lokalisierten Begriff und einer Beschreibung zugeordnet ist. Der Begriff wird unterstrichen angezeigt. Wenn Sie den Mauszeiger über den Begriff bewegen, wird der Hilfe-Cursor angezeigt, und wenn Sie mit der Maus darüber fahren, wird die Beschreibung als Tooltip angezeigt. Auf Touchscreen-Geräten wird durch Antippen des Begriffs eine Popup-Definition angezeigt.

## Glossarbegriffe verwalten

Sie können die Sammlung der Glossarbegriffe im CMS bearbeiten.

1. Klicken Sie auf das Symbol **Hinzufügen** ( **+** ), um einen neuen **Schlüssel** hinzuzufügen.
2. Bearbeiten Sie den Schlüssel und klicken Sie dann auf das Symbol **Hinzufügen** ( **+** ), um eine neue **Übersetzung** hinzuzufügen.
3. Bearbeiten Sie die Übersetzung.
4. Weisen Sie ihm einen **Sprachcode** zu.
5. Geben Sie ein, wie der **Begriff** im Text erscheinen soll.
6. **Geben Sie die Definition ein.**

Der Glossarbegriff verwendet die Sprachversion der Seite, auf der er angezeigt wird. Wenn keine Definition vorhanden ist, wird auf die Ausgangssprache zurückgegriffen.

## Verwendung von Glossarbegriffen

1. Wählen Sie **Glossarbegriff** aus der Liste **Einbetten**.
2. Bearbeiten Sie die Komponente.
3. Geben Sie den **Schlüssel** ein.
