# Registerkarten

Mit der Komponente **Tabs** können Sie Ihrem Thema Inhalte mit Registerkarten hinzufügen.

1. Wählen Sie **Tabs** aus der Liste **Einbetten** aus.
2. Bearbeiten Sie die Komponente.
3. Für jede Registerkarte:
   1. Wählen Sie **Tab Item** aus der Liste **Einbetten**.
   2. Geben Sie einen **Wert** ein.
   3. Geben Sie einen **Titel** ein.
   4. Geben Sie den **Inhalt** ein.
   5. (Optional) Legen Sie diese Registerkarte als **Standard** fest.
4. **Speichern** Sie Ihre Änderungen.

<Admonition type="danger" title="Warnung">
  Das CMS kann beim Bearbeiten von Registerkarten den Kontext verlieren. Sie sollten daher häufig speichern. Das CMS kann in eine Situation geraten, in der es den Kontext der Registerkarte nicht finden kann. Klicken Sie in diesem Fall auf das Symbol **Raw Markdown** und nehmen Sie die Änderungen direkt im Markdown vor. Speichern Sie die Änderungen und kehren Sie dann zum Rich-Text-Editor zurück.
</Admonition>

#### Beispiel:

<Tabs>
  <TabItem label="Apple" value="apple">
    Dies ist ein Apfel 🍎
  </TabItem>

  <TabItem label="Orange" value="orange">
    Dies ist eine Orange 🍊
  </TabItem>

  <TabItem label="Banana" value="banana">
    Dies ist eine Banane 🍌
  </TabItem>
</Tabs>

Weitere Informationen finden Sie unter [Tabs](https://docusaurus.io/docs/markdown-features/tabs) in der Docusaurus-Dokumentation.
