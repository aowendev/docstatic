# Wikis

Eine Wiki-Seite ist eine Web-/Dokumentationsseite, die für die Zusammenarbeit konzipiert ist. Sie lassen sich leicht erstellen und bearbeiten und werden häufig für den internen Wissensaustausch im Team sowie für öffentlich zugängliche Hilfeinhalte verwendet.

Bei der Arbeit mit Wiki-Themen im CMS:

* Erstellen Sie Wiki-Themen im Ordner `/docs/wiki/`.
* Sie müssen die Wiki-Themen nicht zum Inhaltsverzeichnis hinzufügen. Die Seitenleiste für Wiki-Themen wird automatisch auf der Grundlage der Dateistruktur im Repository generiert.
* Wiki-Themen enthalten keine Metadaten.

DocStatic kann eine Kategorie automatisch mit ihrem Indexdokument verknüpfen. Ein Kategorie-Indexdokument ist ein Dokument, das dieser Dateinamenskonvention folgt:

Benannt als `README` (Groß-/Kleinschreibung spielt keine Rolle): `docs/wiki/folder/readme.mdx`.

Weitere Informationen finden Sie unter [Autogenerated](https://docusaurus.io/docs/sidebar/autogenerated#autogenerated-sidebar-metadata) in der Docusaurus-Dokumentation.

## Erstellen einer Wiki-Seite im CMS

1. Wählen Sie die Sammlung **Wiki Pages** aus.
2. Klicken Sie auf **Datei hinzufügen**.

<Abbildung img="/img/docs/wiki-pages-add-wiki.png" caption="TinaCMS. Ein Callout hebt die Option „Wiki” und die Option „Datei hinzufügen” hervor." />

1. Geben Sie Ihrer neuen Wiki-Seite einen **Titel**.
2. Geben Sie den Inhalt ein.
3. Klicken Sie auf **Speichern**.
