# Einführung in die Dokumentation

Die Dokumentationsfunktion bietet Ihnen die Möglichkeit, Markdown-Dateien in einem hierarchischen Format zu organisieren.

<Admonition type="info" title="Info">
  Eine vollständige Liste der Optionen finden Sie in der Docusaurus-Dokumentation unter [Docs Plugin API Reference](https://docusaurus.io/docs/api/plugins/@docusaurus/plugin-content-docs).
</Admonition>

Informationen zum Konfigurieren des Nur-Dokumente- oder Nur-Blog-Modus finden Sie unter [Einführung in die Dokumentation](https://docusaurus.io/docs/docs-introduction) in der Docusaurus-Dokumentation.
