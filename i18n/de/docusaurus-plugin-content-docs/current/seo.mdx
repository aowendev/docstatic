DocStatic unterstützt die Suchmaschinenoptimierung auf verschiedene Weise:

* Globale Metadaten
* Für Menschen lesbare Links
* Bild-Metabeschreibung
* Umfangreiche Suchinformationen
* Robots-Datei
* Metadaten für einzelne Seiten
* Sitemap-Datei
* Strukturierte Inhalte

Weitere Informationen finden Sie unter [Suchmaschinenoptimierung](https://docusaurus.io/docs/seo) in der Docusaurus-Dokumentation.
