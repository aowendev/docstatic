# Internationalisierung

Die Unterstützung für Internationalisierung ([i18n](https://en.wikipedia.org/wiki/Internationalization_and_localization)) wird vom statischen Website-Generator bereitgestellt. Ausführliche Informationen finden Sie unter [i18n – Einführung](https://docusaurus.io/docs/i18n/introduction) in der Docusuarus-Dokumentation.

## Bearbeiten Sie übersetzte Dokumente im CMS

1. Wählen Sie aus der Sammlung **Übersetzungen** das Dokument aus, das Sie bearbeiten möchten.
2. Nehmen Sie Ihre Änderungen vor.
3. Setzen Sie den **Dokumentstatus** je nach Bedarf auf **In Übersetzung** oder **Übersetzung genehmigt**.
4. Klicken Sie auf **Speichern**.
