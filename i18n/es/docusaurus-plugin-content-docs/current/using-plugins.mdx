El núcleo del generador de sitios estáticos no proporciona ninguna función propia. Todas las funciones se delegan a complementos individuales: la función de documentación proporcionada por el [complemento docs](https://docusaurus.io/docs/api/plugins/@docusaurus/plugin-content-docs); la función de blog proporcionada por el [complemento blog](https://docusaurus.io/docs/api/plugins/@docusaurus/plugin-content-docs); o las páginas individuales proporcionadas por el [complemento páginas](https://docusaurus.io/docs/api/plugins/@docusaurus/plugin-content-docs).

DocStatic está preconfigurado para utilizar los complementos necesarios para admitir:

* Ecuaciones matemáticas KaTeX
* Búsqueda Lunr
* Diagramas Mermaid
* Documentación OpenAPI
* Documentación GraphQL

Para obtener más información, consulte [Uso de complementos](https://docusaurus.io/docs/using-plugins) en la documentación de Docusaurus.
