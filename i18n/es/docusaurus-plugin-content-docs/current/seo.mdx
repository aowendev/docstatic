DocStatic admite la optimización para motores de búsqueda de diversas formas:

* Metadatos globales
* Enlaces legibles para humanos
* Metadescripción de imágenes
* Información de búsqueda enriquecida
* Archivo robots
* Metadatos de página única
* Archivo de mapa del sitio
* Contenido estructurado

Para obtener más información, consulte la sección [Optimización para motores de búsqueda](https://docusaurus.io/docs/seo) en la documentación de Docusaurus.
