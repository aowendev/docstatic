# Blog

La función de blog te permite crear artículos con marca de tiempo.

<Admonition type="info" title="Infor">
  Consulta [Referencia de la API del complemento Blog](https://docusaurus.io/docs/api/plugins/@docusaurus/plugin-content-blog) en la documentación de Docusaurus para obtener una lista exhaustiva de opciones.
</Admonition>

1. En la colección **Artículos del blog** del CMS, haz clic en **Añadir archivo**.
2. Dale un **Título**.
3. Añada los **Autores**. Para cada autor, edite el componente y añada:
   1. Su **Nombre**.
   2. Su **Título**.
   3. Una **URL**.
   4. Una **URL de imagen**.
4. Seleccione la **Fecha** de publicación. Los artículos con fecha posterior no aparecerán hasta que se reconstruya el sitio.
5. (Opcional) Añade cualquier **Etiqueta** de taxonomía.
6. Introduce el contenido.
7. Haz clic en **Guardar**.

Para obtener más información, consulta [Blog](https://docusaurus.io/docs/blog) en la documentación de Docusaurus.
