# Comprobación de estilo

Una de las características que más echan en falta los escritores que trabajan con documentos como código es un buen corrector ortográfico, gramatical y de estilo. Por defecto, la comprobación ortográfica la proporciona el navegador web o el entorno de desarrollo integrado. Esto puede ser suficiente para sitios web sencillos, pero si necesitas algo más avanzado, necesitarás un corrector gramatical específico.

## LanguageTool

DocStatic se ha probado con [LanguageTool](https://languagetool.org/), un corrector gramatical basado en inteligencia artificial que admite más de 20 idiomas. El [registro](https://languagetool.org/register) para el plan básico es gratuito. El [plan premium](https://languagetool.org/premium) añade compatibilidad con guías de estilo para particulares y equipos. Hay disponibles planes [empresariales](https://languagetool.org/sales) personalizados. También hay un [descuento](https://languagetool.org/premium) para estudiantes y organizaciones sin ánimo de lucro.

## Autoalojado

Dado que LanguageTool es de código abierto, tienes la opción de [autoalojarlo](https://languagetool.org/dev). En este caso, no hay cuotas de suscripción, pero necesitarás un equipo de desarrollo para configurar tu instancia de LanguageTool y tendrás que proporcionar tu propio soporte.
