Con Apple iPadOS, puedes convertir cualquier página web en una aplicación web que se ejecuta en su propia instancia de Safari con su propio icono en la pantalla de inicio. Puedes utilizar esta función para crear una aplicación para el CMS alojado en la nube.

1. Inicia sesión en el CMS.
2. Navega hasta la colección que deseas utilizar como página de inicio. Por ejemplo, la colección **Temas** o el **Índice**.
3. Pulsa el icono **Compartir**, pulsa **Más** ( **…** ) y pulsa **Añadir a la pantalla de inicio** ( **+** ).
4. Dale un nombre a la aplicación. Por defecto, utilizará TinaCMS.
5. Haz clic en **Añadir**.
