# Actualización

Aunque DocStatic aún se encuentra en una fase inicial de desarrollo, los cambios deben fusionarse individualmente desde el repositorio principal. Sin embargo, los componentes Node.js pueden actualizarse de forma segura sustituyendo el archivo [package.json](https://github.com/aowendev/docstatic/blob/main/package.json) por la última versión del repositorio principal. Después de la actualización, ejecute:

```shell
yarn install
```
