# Advertencias

El componente **Advertencias** le permite añadir advertencias con estilo.

1. Seleccione **Advertencia** en la lista **Incrustar**.
2. Edite el componente y seleccione un **Tipo** (**Nota**, **Consejo**, **Información**, **Precaución** o **Advertencia**).
3. Asigne un **Título**.
4. Introduzca el **Contenido**.

#### Ejemplos:

<Admonition type="note" title="Nota">
  Utilice advertencias de tipo nota para proporcionar información complementaria.
</Admonition>

<Admonition type="tip" title="Consejo">
  Utilice las advertencias de tipo consejo para proporcionar consejos que se pueden ignorar sin problema. 
</Admonition>

<Admonition type="info" title="Información">
  Utilice las advertencias de tipo información para proporcionar información lo suficientemente importante como para que destaque del resto del texto. 
</Admonition>

<Admonition type="caution" title="Precaución">
  Utilice advertencias de precaución cuando algo pueda impedir que se complete una acción prevista. 
</Admonition>

<Admonition type="danger" title="Advertencia">
  Utilice advertencias de advertencia cuando exista riesgo de lesiones, violación de la seguridad o pérdida de datos. 
</Admonition>
