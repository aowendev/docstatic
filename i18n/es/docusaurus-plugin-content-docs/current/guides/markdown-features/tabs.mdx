# Pestañas

El componente **Pestañas** le permite añadir contenido con pestañas a su tema.

1. Seleccione **Pestañas** en la lista **Incrustar**.
2. Edite el componente.
3. Para cada pestaña:
   1. Seleccione **Elemento de pestaña** en la lista **Insertar**.
   2. Asigne un **Valor**.
   3. Asigne un **Título**.
   4. Introduzca el **Contenido**.
   5. (Opcional) Establezca la pestaña como **Predeterminada**.
4. **Guarde** los cambios.

<Admonition type="danger" title="Advertencia">
  El CMS puede perder el contexto al editar pestañas. Debe guardar con frecuencia. El CMS puede llegar a una situación en la que no pueda encontrar el contexto de la pestaña. En este caso, haga clic en el icono **Raw Markdown** y realice los cambios directamente en Markdown. Guarde y vuelva al editor de texto enriquecido.
</Admonition>

#### Ejemplo:

<Tabs>
  <TabItem label="Apple" value="apple">
    Esto es una manzana 🍎
  </TabItem>

  <TabItem label="Orange" value="orange">
    Esto es una naranja 🍊
  </TabItem>

  <TabItem label="Banana" value="banana">
    Esto es un plátano 🍌
  </TabItem>
</Tabs>

Para obtener más información, consulta [Pestañas](https://docusaurus.io/docs/markdown-features/tabs) en la documentación de Docusaurus.
