# Términos del glosario

El componente **Término del glosario** le permite introducir una clave que se asigna a un término y una descripción localizados. El término se muestra subrayado. Al mover el cursor sobre el término, aparece el cursor de ayuda, y al pasar el cursor por encima, se muestra la descripción como una información sobre herramientas. En los dispositivos con pantalla táctil, al pulsar sobre el término se muestra una definición emergente.

## Gestión de términos del glosario

Puede editar la colección de términos del glosario en el CMS.

1. Haga clic en el icono **Añadir** ( **+** ) para añadir una nueva **Clave**.
2. Edite la clave y, a continuación, haga clic en el icono **Añadir** ( **+** ) para añadir una nueva **Traducción**.
3. Edite la traducción.
4. Asigne un **Código de idioma**.
5. Introduzca cómo debe aparecer el **Término** en el texto.
6. **Introduzca la definición.**

El término del glosario utilizará la versión lingüística de la página en la que se muestra. Si no existe ninguna definición, se utilizará el idioma de origen.

## Uso de términos del glosario

1. Seleccione **Término del glosario** en la lista **Incrustar**.
2. Edite el componente.
3. Introduzca la **Clave**.
