# Metadatos del encabezado

DocStatic establece automáticamente metadatos útiles para la página en `<html>`, `<head>` y `<body>`.
Es posible añadir metadatos adicionales (o sobrescribir los existentes) con la etiqueta `<head>` en los archivos Markdown.

En el CMS, esto debe añadirse en el editor Markdown. No se reconocerá si lo envuelve en el componente **Passthrough**.

Para obtener más información, consulte [Metadatos de encabezado](https://docusaurus.io/docs/markdown-features/head-metadata) en la documentación de Docusaurus.
