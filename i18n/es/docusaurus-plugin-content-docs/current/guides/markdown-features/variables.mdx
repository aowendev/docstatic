# Conjuntos de variables

El componente Variable le permite colocar una variable localizada en su texto.

## Administrar conjuntos de variables

1. En el CMS, desde la colección **Conjuntos de variables**, añada un nuevo conjunto o edite uno existente.
2. Introduzca o edite el **Nombre del conjunto de variables**.
3. Añada o edite una **Variable**.
4. Introduzca o edite la **Clave**.
5. Añada o edite una **Traducción**.
6. Edite o introduzca un par **Idioma** y **Valor**.

## Usar variables

1. Seleccione el componente **Variable** de la lista **Incrustar**.
2. Edite el componente.
3. Seleccione las claves de la **Variable** (**Conjunto de variables**).

#### Ejemplo

<VariableSet variableSelection="writing-terms|single-sourcing" initcap={true} />
