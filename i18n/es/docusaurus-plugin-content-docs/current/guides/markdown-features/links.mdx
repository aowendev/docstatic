# Enlaces Markdown

Hay dos formas de añadir un enlace a otra página: a través de una ruta URL o una ruta de archivo.

```
- [Ruta URL a otro documento](./installation)
- [Ruta de archivo a otro documento](./installation.mdx)
```

En el CMS, puedes añadir un enlace desde el portapapeles seleccionando una sección de texto y pegándola.

También puede resaltar una sección de texto y hacer clic en el icono Enlace para añadir una ruta.

O bien, puede colocar el cursor donde desee que aparezca el enlace, hacer clic en el icono e introducir la ruta y el texto que se mostrará.

Para obtener más información sobre los enlaces, consulte [Enlaces Markdown](https://docusaurus.io/docs/markdown-features/links) en la documentación de Docusuarus.
