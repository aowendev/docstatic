# Complementos MDX

MDX tiene un [sistema de complementos](https://mdxjs.com/advanced/plugins/) integrado que se puede utilizar para personalizar la forma en que se analizan y transforman los archivos Markdown a JSX. Sin embargo, si cambia el comportamiento de Markdown, también debe reflejar estos cambios en el CMS.

<Admonition type="danger" title="Advertencia">
  Los complementos MDX que cambian el comportamiento de los componentes Markdown estándar pueden romper la compatibilidad con el CMS. Proceda bajo su propia responsabilidad.
</Admonition>

Para obtener más información, consulte [Complementos MDX](https://docusaurus.io/docs/markdown-features/plugins) en la documentación de Docusaurus.

<Admonition type="note" title="Nota">
  DocStatic preinstala y configura los [complementos matemáticos](math-equations) por usted.
</Admonition>
