# Introducción a los documentos

La función de documentos le ofrece una forma de organizar los archivos Markdown en un formato jerárquico.

<Admonition type="info" title="Información">
  Consulte la [Referencia de la API del complemento Docs](https://docusaurus.io/docs/api/plugins/@docusaurus/plugin-content-docs) en la documentación de Docusaurus para obtener una lista exhaustiva de opciones.
</Admonition>

Para obtener información sobre cómo configurar los modos solo documentos o solo blog, consulte la [Introducción a los documentos](https://docusaurus.io/docs/docs-introduction) en la documentación de Docusaurus.
