# Internacionalización

El generador de sitios estáticos proporciona soporte para la internacionalización ([i18n](https://en.wikipedia.org/wiki/Internationalization_and_localization)). Para obtener información detallada, consulte [i18n - Introducción](https://docusaurus.io/docs/i18n/introduction) en la documentación de Docusuarus.

## Editar documentos traducidos en el CMS

1. En la colección **Traducciones**, seleccione el documento que desea editar.
2. Realice los cambios necesarios.
3. Establezca el **Estado del documento** en **En traducción** o **Traducción aprobada**, según sea necesario.
4. Haga clic en **Guardar**.
