# Configuración

La configuración del sitio DocStatic se gestiona a través del archivo `docusaurus.config.ts`. Para obtener información sobre cómo editar este archivo directamente, consulte [Configuración](https://docusaurus.io/docs/configuration) en la documentación de Docusaurus.

Para utilizar el CMS para editar la configuración, vaya a la colección **Configuración**. Desde aquí, puede configurar lo siguiente del sitio:

* **Título**.
* **Eslogan**.
* **Favicon**.
* **URL del esquema GraphQL**.
* **Modo de color** predeterminado (oscuro o claro).
* Elementos del **pie de página**.
* **Idiomas**.
* **Logotipo**.
* Esquemas **OpenAPI**.
* **Resaltado de sintaxis**.
* **URL**.
* **Barra de navegación**.
* Ajustes de la **barra lateral**.
* Ajustes del **blog**.

Hay una colección **Tema** independiente para cambiar la combinación de colores y los ajustes tipográficos.
