# My Site

This is a documentation site built with [docStatic](https://github.com/aowendev/docstatic).

## Getting Started

1. Install dependencies: `npm install`
2. Start development server: `npm run dev`
3. Open [http://localhost:3000](http://localhost:3000)

## Features

- Built with Docusaurus and TinaCMS
- Markdown and MDX support
- Rich text editor
- Git-based workflow
- Search functionality
- Responsive design

## Configuration

Edit the configuration files in the `config/` directory to customize your site.

## Deployment

Build the site: `npm run build`

Serve locally: `npm run serve`
