# Internationalization

Internationalization ([i18n](https://en.wikipedia.org/wiki/Internationalization_and_localization)) support is provided by the static site generator. For detailed information, refer to [i18n - Introduction](https://docusaurus.io/docs/i18n/introduction) in the Docusuarus documentation.

## Edit translated docs in the CMS

1. From the **Translations** collection, select the document you want to edit.
2. Make your changes
3. Set the **Document Status** to **In Translation** or **Translation Approved** as required.
4. Click **Save**.
