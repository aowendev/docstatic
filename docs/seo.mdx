DocStatic supports search engine optimization in various ways:

* Global metadata
* Human readable links
* Image meta description
* Rich search information
* Robots file
* Single page metadata
* Sitemap file
* Structured content

For more information, refer to the [Search engine optimization](https://docusaurus.io/docs/seo) in the Docusaurus documentation.
