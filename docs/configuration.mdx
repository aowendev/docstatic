# Configuration

DocStatic site configuration is managed through the `docusaurus.config.ts` file. For information about editing this file directly, refer to [Configuration](https://docusaurus.io/docs/configuration) in the Docusaurus documentation.

To use the CMS to edit the configuration, navigate to the **Settings** collection. From here, you can configure the site's:

* **Title**.
* **Tagline**.
* **Favicon**.
* **GraphQL Schema URL**.
* Default **Color Mode** (dark or light).
* **Footer** items.
* **Languages**.
* **Logo**.
* **OpenAPI** schemas.
* **Syntax Highlighting**.
* **URL**.
* **Navbar**.
* **Sidebar** settings.
* **Blog** settings.

There is a separate **Theme** collection for changing the color scheme and typography settings.
