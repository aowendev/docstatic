# Docs introduction

Use **Docs** to organize Markdown files in a hierarchical format, like the structure of a user guide or a table of contents.

<Admonition type="info" title="Info">
  Refer to [Docs Plugin API Reference](https://docusaurus.io/docs/api/plugins/@docusaurus/plugin-content-docs) in the Docusaurus documentation for an exhaustive list of options.
</Admonition>

For information on configuring docs-only or blog-only modes, refer to [Docs introduction](https://docusaurus.io/docs/docs-introduction) in the Docusaurus documentation.
