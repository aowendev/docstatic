# Variable sets

The Variable component enables you to place a localized variable in your text.

## Manage variable sets

1. In the CMS from the **Variable Sets** collection, add a new set or edit an existing set.
2. Enter or edit the **Variable Set Name**.
3. Add or edit a **Variable**.
4. Enter or edit the **Key**.
5. Add or edit a **Translation**.
6. Edit or enter a **Language** and **Value** pair.

## Use variables

1. Select the **Variable** component from the **Embed** list
2. Edit the component.
3. Select the **Variable** (**Variable Set**) keys.
