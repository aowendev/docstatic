# Head metadata

DocStatic automatically sets useful page metadata in `<html>`, `<head>` and `<body>` for you.
It is possible to add extra metadata (or override existing ones) with the `<head>` tag in Markdown files.

In the CMS, this must be added in the Markdown editor. It will not be recognized if you wrap it in the **Passthrough** component.

For more information, refer to [Head metadata](https://docusaurus.io/docs/markdown-features/head-metadata) in the Docusaurus documentation.
